Capacity

Each Anthropic subscription account has a 5-hour rolling token window — a soft limit on how many tokens can be consumed in any 5-hour period. This page shows how much of that window is used and how long until accounts need to "rest." Numbers are estimates: the API only reports utilisation %, not exact token counts.
